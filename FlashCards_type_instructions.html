Flash Cards/Type

Instructions

Overview

The main difference between the Flash Card game and the Flash Card/Type version of the game is that now you can type in the answer, and, similar to the Multiplication game, it keeps track of how many you got right.

Let's just talk about what we changed between the two games:

NewCard()

We added two more lines to the code that writes the front of the flash card:

var FC = "<div class='card front' >" + FlashCards[i][0]; 
FC += "<br/><textarea id='A' rows='1' cols='20'></textarea>";
FC += "<br/><button type='button' onclick='FlipCard(" + i + ")'>Submit</button></div>";

We added a text area, for you to type in the answer, and a Submit button — just as in the Hallo YOU game.

What is FC += ...?

Can you guess? Right, it just short-hand for FC = FC + ...

We also added the command:

document.getElementById("A").value = "";

at the end of the NewCard() function. We need that to erase the answer you typed from the previous card...

FlipCard()

Here, we read what you typed, again similar to how we did it in the Hallo YOU game, then we check if what you typed is the same as what is on the back of the card with:

if ( A == FlashCards[i][1] ) { ... }

and keep track of your score in the same way we did in the Multiplication game.
