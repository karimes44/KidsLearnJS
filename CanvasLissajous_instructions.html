Canvas Draw/Lissajous Figures

Instructions

Check out Lissajous Curves at Wikipedia.

Then draw some by setting A and B to some numbers between 1.0 and maybe 5.0 (try decimal fractionslike 2.3 also). Try setting δ bewteen 0 and 90. If the figure is not closed, you can also increase the number of loops.

You can draw several different figures on-top of each other by changing the A, B, or δ values and clicking Submit again.

Try it - makes pretty looking figures.
