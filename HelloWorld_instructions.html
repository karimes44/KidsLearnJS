Instruction

Have you written a text in Google Docs, or MS Word, before?

With these tools you first write, say, your title, and then highlight the title, and select from the top toolbar, maybe that it should be bold and with a font size of 20pt. So you directly tell the computer how a particular word should be displayed.

Writing a text with HTML is a bit different...

With HTML, you put "tags" around everything you write to "mark-up" the text. There mark-ups tell the computer what this piece of text is, but not how to display it.

With another tool, called CSS, you tell the computer how text that belongs to a certain mark-up type should be displayed.

This has the advantage that you can write your text first, and then later think how to make it look pretty. Also, if you change your mind you can, for example change the appearance of all tags in the entire document with just one CSS instruction.
