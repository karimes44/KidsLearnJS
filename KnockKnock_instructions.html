Knock Knock

Instruction

With just a few changes to the Hello You game, we can make a Knock, Knock joke game!

  1. In this game the function KnockKnock(i) has an argument: i. We use this counter to step through the lines of the joke.
  2. The command <span id='Dialog'></span> section now contains all the text. The KnockKnock(i) function can replace the entire text.
  3. At the beginning in the Dialog area the button calls KnockKnock(1), so the argument is 1.
  4. In the KnockKnock(i) function, we check if the variable i is 1. If so, we write back to the Dialog area:
    TEXT = "<h1>Boo</h1>" + '<button type="button" onclick="KnockKnock(2)">Boo Who?</button>'
    ... so we are creating a new button which will call KnockKnock(2), i.e, with the argument 2.
  5. ... now that will call the same KnockKnock(i) function, but something else will happen as a result! We write back:
    TEXT = "<h1>Please don't cry!</h1>";.

Challenge

Can you add more else if blocks, so that the game continues to tell more "Knock, knock" jokes?

Here are a few examples, but I bet you know even better ones!

Knock, knock.

Anna Partridge.
