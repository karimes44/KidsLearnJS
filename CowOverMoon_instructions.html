The Cow Flies Over The Moon

Instruction

We focused a lot on JavaScript, and used only basic CSS, like setting font sizes and colors.

But here just a quick example of some of the cool things you can do with CSS: You can move things around! Here we move one picture with respect to another. We downloaded from the web a picture of a landscape with a full moon, and one of a cow. The cow picture came on a "transparent" background, that comes in handy when we overlay the pictures.

<body> and <h1>

We set the background color to navy blue and the color in which the heading is written to white:

<style>
 body { background-color: navy; }
 h1   {  color:          white; }
</style>

<div> and <img>

We load two images with the <img> tag. To learn what the different attributes mean, check out: W3School: HTML Images

We place the images into a <div> tag, which is just a generic tag, so that we can do something with it. We give the identifiers "moon" and "cow".

<style>

In the CSS style section, we refer to these identifiers with #. (See W3School: CSS Syntax and Selectors).

To be able to move the #cow with respect to the #moon, we have to define one with position: relative; and the other one with position: absolute;.
To learn more about that, check out: W3School: CSS Layout - The position Property.

In the #cow section we add an animation that we call mymove. We state that it should last for 5 seconds and repeat forever (infinite).
See W3School: CSS Animations.

The mymove animation is defined in the @keyframes section, where we specify the position of the cow during the animation.
Check out: W3School: CSS @keyframes Rule.

Sharing

The two images are in different files, so if you want to send this to a friend you need to send the images as well. Also, your friend will have to save the HTML file and the images first in the same directory, and then load the HTML file into the browser. Starting it from the email attachment will not load the images...
