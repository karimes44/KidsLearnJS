Multiplication Table Game

Instruction

Need a little help learning your multiplication tables? Let the computer give you a hand... but first you have to give the computer a hand and program the Multiplication Table Game!

This game with ask you randomly a multiplication table question from 1x1 to 12x12. You can then type in the answer. A message window will tell you if you got it right. The computer also keeps score on how many questions you got right.

For writing this program we will mainly re-use things from the previous games we created.

<body>

  1. We use a table with two cells, but just to nicely align the question and the textarea where you can type in the answer.
  2. Once you typed in the answer, you click the Submit button, which will call the Answer() function.
  3. The score will be displayed in the paragraph with id="Score"

<script>

We define three global variables:

  1. z will contain the correct answer
  2. nQ will count the number of questions
  3. nC will count the number of right answers

PoseQuestion()

  1. We create two random integer numbers and store them in the variables X and y. Because we need the numbers to be between 1 and 12, and not 0 and 11, we use Math.ceil, which will turn 3.1415 into 4, and not Math.floor, which would turn 3.1415 into 3.
  2. We multiply the numbers — and write the result in the table cell with id="Question".

Answer()

  1. We read the value you types into the textarea with:
    var Z = document.getElementById("Z").value;
    (Note how z and Z are different variable! Javascript is "case-sensitive"!)
  2. We check if the correct answer (the value of z) is the same number as the one you typed (value of Z).
  3. If so, we increase the counter nC by one and alert you through a message pop-up that you got is right!
    What is nC++;? That's just short for nC = nC + 1;.
  4. If not, we alert you about the mistake and show you the correct answer.
  5. We increase the questions counter, again with nQ++;, and call the function ShowScore() that shows the current score.
  6. Lastly, we pose the next question.

window.onload

In our previous games, you started the game by clicking something. In the multiplication game, however, the game starts as soon as the game (or window) is loaded. With the command:

window.onload = PoseQuestion;

... we tell the computer to call the function PoseQuestion as soon as has loaded the game.
