SVG Graphics

The flow diagram above is created with another feature of HTML, called "Scalable Vector Graphics" or SVG. We will touch on SVG in the Keep the Beat game, however, you can see already here how it is written by loading this HTML file into your Caret Editor or pressing Ctrl-U in the browser window (or right mouse click: View Page Source).

For a quick introduction to SVG take a look at: W3School: HTML SVG, or for a complete (but hard to read) command reference see: Mozilla: SVG
