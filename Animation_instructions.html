Instructions

In the code you'll see a sequence of "frames", like:

["0000000000",
 "0000660000",
 "0006006000",
 "0060000600",
 "0600000060",
 "0600000060",
 "0060000600",
 "0006006000",
 "0000660000",
 "0000000000"],

Just overwrite the numbers and symbols with a "0" for black, or a color code like 1,2,3,4,5,6,7,8,9,A, to form a pattern or picture. The frame above, for example shown a circle in the color code "6", which in the style section is mapped to #ff00ff (Check out the Color Picker to see what color that is).

If you design a good sequence the Animation Loop will show a "movie clip".
