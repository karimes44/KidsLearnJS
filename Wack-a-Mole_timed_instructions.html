Wack-a-Mole (timed)

Instructions

Overview

To make it a "real" game we need to add goal, or a challenge!
How about "Wack as many moles as you can in 20 sec" ?

For that we need just a few changes ... first, we add two global variables:

var GameTime = 20;
var Nw = 0;

GameTime defines how long the game is played (here 20 seconds), and Nw is the counter for the number of wacks. After each wack we add one to the counter with Nw++;

Play()

If iprevious == -1, that is, we are just starting the game, we first remove the button, so you can't start the game again while it is already going. This we do with:

document.getElementById('BUTTON').innerHTML = "";

Then we do something new: We start a timer that, after 20 seconds will call the GameUp function:

STOP = setTimeout(GameUp, GameTime*1000)

You can learn more about this function at: W3School: JS Timing

GameUp()

This function pops up an alert message, that also shows how many moles you wacked, and then erases the game board, so that you cannot keep playing anymore without restarting the game.
